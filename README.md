Onegai-oyodosan (おねがい！大淀さん)
===============

This is an automation tool about Kancolle.

Usage
=========
  - Download [Sikuli]
  - Create a folder named oyodosan.sikuli and put in
  - Excucate Sikuli script with IDE


Required
========
  - All fleet open
  - Reach Level 3-2

[sikuli]:http://www.sikuli.org/
